Enhance my existing Node.js + Express + frontend app to create an advanced "Download Printable Worksheets" system inside the Parent Hub.

GOALS:

* Limit downloads to max 5 files per day per user
* Once a file is downloaded, do NOT show it again to that user
* Show worksheets only for parents of children aged 2–7 years
* Add categories (coloring, math, tracing, etc.)
* Improve UI (download button visibility, preview, fallback logo)

BACKEND:

1. Update /worksheets API:

   * Fetch files recursively from Google Drive (including subfolders)
   * Return:
     {
     id,
     name,
     category,
     downloadUrl,
     previewUrl
     }

2. CATEGORY LOGIC:

   * Infer category from file name or folder name:

     * "color" → coloring
     * "math" → math
     * "trace" → tracing

3. DAILY LIMIT SYSTEM:

   * Track downloads per user (use localStorage or userId)
   * Max 5 downloads per day
   * Reset count every 24 hours

4. HIDE DOWNLOADED FILES:

   * Store downloaded file IDs in localStorage
   * Filter them out from API response

5. AGE FILTER:

   * Only show this section if user child age is between 2–7
   * Age can come from user profile or mock variable

5. DOWNLOAD BUTTON:
